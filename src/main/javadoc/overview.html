Welcome to the Java documentation of the Keypop Calypso Crypto Legacy SAM Java API, part of the Keypop open-source project hosted by the Eclipse Foundation.

Aligned with the Terminal Calypso Crypto Legacy SAM UML specifications proposed by the Calypso Networks Association (available here), it defines the interfaces needed to operate a transaction with a Calypso legacy SAM (SAM C1, HSM, etc.) and also to manage secure card transactions.

Note: as you navigate, you might observe that certain interfaces either reference or extend elements from the Keypop Reader Java API and the Keypop Calypso Card Java API.
